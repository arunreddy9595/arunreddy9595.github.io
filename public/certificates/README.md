# Certificates Directory

Place your certificate PDF files here with the following names:

1. `azure-fundamentals.pdf` - Azure Fundamentals (AZ-900) certificate
2. `comptia-a-plus.pdf` - CompTIA A+ certificate
3. `fortinet-nse1.pdf` - Fortinet NSE 1 certificate

These files will be used by the certifications section on your portfolio website.
